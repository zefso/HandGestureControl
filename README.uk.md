Ласкаво просимо до **Системи Управління Жестами Рук**! 🖱️👋
Цей проект дозволяє вам керувати мишею комп'ютера та виконувати команди за допомогою простих жестів рук. Використовуючи **Комп'ютерний Зір** та **Глибоке Навчання**, він надає безконтактний інтерфейс взаємодії.

Побудовано на основі потужних технологій:
-   **MediaPipe** для надшвидкого відстеження рук.
-   **LSTM (Long Short-Term Memory)** нейронних мереж для точного розпізнавання жестів.
-   **OpenCV** для обробки зображень.

## ✨ Особливості

-   **🖐️ Відстеження в реальному часі**: Миттєво виявляє та відстежує ключові точки рук з високою точністю.
-   **🧠 Розумне розпізнавання**: Використовує модель LSTM для розуміння динамічних жестів (послідовностей рухів), а не лише статичних поз.
-   **🖱️ Керування Мишею**: Переміщуйте курсор, клікайте, перетягуйте та прокручуйте за допомогою природних рухів рук.
-   **⚡ Низька затримка**: Оптимізовано для плавної роботи на стандартних процесорах.
-   **🎨 Власні жести**: Постачається з попередньо навченими жестами, але ви можете легко додати свої!
    -   `static` 🏋️ (Без дії)
    -   `swipe_right` ➡️ (Наступне вікно/вкладка)
    -   `swipe_left` ⬅️ (Попереднє вікно/вкладка)
    -   `ok` 👌 (Пілтвердження/Клік)
    -   `stop` 🤚 (Пауза/Утримання)
    -   `browser` ✌️ (Відкрити Браузер)
    -   `fist_left` ✊ (Хапати/Перетягувати)

## 🚀 Встановлення

### Вимоги

-   **Python 3.8** або новіше.
-   Робоча **Веб-камера**.
-   **Git** (необов'язково, для клонування).

### Кроки Налаштування

1.  **Клонуйте Репозиторій**
    Відкрийте термінал або командний рядок і виконайте:
    ```bash
    git clone <адреса_репозиторію>
    cd HandGestureControl
    ```

2.  **Створіть Віртуальне Середовище** (Рекомендовано)
    Найкраще тримати залежності ізольованими.

    *   **Windows:**
        ```bash
        python -m venv venv
        venv\Scripts\activate
        ```
    *   **macOS / Linux:**
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```

3.  **Встановіть Залежності**
    Встановіть усі необхідні бібліотеки за допомогою `pip`:
    ```bash
    pip install -r requirements.txt
    ```
    > **Примітка:** Основні бібліотеки включають `opencv-python`, `mediapipe`, `torch`, `pyautogui`, `scikit-learn`.

## 🕹️ Використання

### 1. Збір Даних (Необов'язково)
*Пропустіть це, якщо хочете використовувати лише попередньо навчену модель.*
Щоб навчити систему на **ваших** специфічних рухах:
```bash
python src/smart_collector.py
```
-   Дотримуйтесь підказок на екрані.
-   Натисніть `n` для зміни жесту.
-   Натисніть `Space` для запису послідовності.
-   Дані зберігаються в `data/<назва_жесту>/`.

### 2. Навчання Моделі
Після збору даних (або додавання нових жестів) перенавчіть "мозок" системи:
```bash
python src/train.py
```
-   Скрипт автоматично завантажує дані, навчає LSTM та перевіряє точність.
-   Найкраща модель зберігається як `models/gesture_lstm_best.pth`.
-   Перевірте `diagram/` для графіків точності/втрат! 📈

### 3. Запуск Системи
Готові керувати мишею?
```bash
python src/main.py
```
-   **Керування**:
    -   **Рух Миші**: Вказуйте вказівним пальцем.
    -   **Лівий Клік**: Зведіть Вказівний + Великий пальці.
    -   **Правий Клік**: Зведіть Середній + Великий пальці.
    -   **Прокрутка**: Підніміть Вказівний + Середній пальці разом.
-   **Вихід**: Натисніть `q`, щоб вийти з програми.

## ⚙️ Конфігурація

Налаштуйте систему в `src/config.py`:

| Змінна | Опис | За замовчуванням |
| :--- | :--- | :--- |
| `GESTURES` | Список назв жестів для розпізнавання | `['static', 'swipe_right', ...]` |
| `SEQ_LENGTH` | Кількість кадрів на послідовність жесту | `30` |
| `THRESHOLD` | Необхідна впевненість для дії | `0.92` (92%) |
| `EPOCHS` | Кількість ітерацій навчання | `150` |

## ❓ Вирішення Проблем

**П: Система не бачить моїх рук.**
> **В:** Переконайтеся, що освітлення хороше. Уникайте контрового світла. Тримайте руки в кадрі.

**П: Курсор миші смикається.**
> **В:** Збільште параметр `smoothing` у `mouse_controller.py` або покращіть освітлення.

**П: Помилка: `Module not found`.**
> **В:** Переконайтеся, що ви активували віртуальне середовище та виконали `pip install -r requirements.txt`.

**П: Жести плутаються.**
> **В:** Спробуйте перенавчити модель на власних даних за допомогою `smart_collector.py`. Руки у всіх рухаються по-різному!
